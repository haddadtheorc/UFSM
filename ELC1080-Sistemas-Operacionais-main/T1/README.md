## Trabalho 1 - Simulador de Processador - Sistemas Operacionais

O programa constrói todos os elementos base até agora para a criação do processador. Há uma fita de memória para armazenar inteiros, suas devidas funções de manipulação e seus tratamentos de erro. Há uma classe independente englobada pela CPU, que controla todas as informações do estado interno da CPU. As instruções base - até agora 20 - estão num .txt para serem lidas pelo programa e facilitar sua manipulação. Ambos estado interno e output do io são mostrados durante execução, bem como salvos em um .txt para análise futura.
